📖 Guide d'Utilisation & Documentation

1. Présentation de l'outil

Ce programme permet de visualiser, filtrer et analyser de manière interactive les sorties de variants structuraux (SV) somatiques générées par l'algorithme Manta et annotées avec AnnotSV.

2. Interprétation des données de VAF

La fréquence allélique de la variante (VAF) est extraite dynamiquement pour chaque échantillon à partir des formats VCF :

  • Format PR:VAF : Paired-end reads support. Représenté par REF,ALT:VAF (ex: 68,1:0.0144).
  • Format PR:SR:VAF : Paired-end et Split-reads support. Représenté par PR_REF,PR_ALT:SR_REF,SR_ALT:VAF (ex: 191,5:133,2:0.0211).
  • La VAF est calculée comme : ALT_TOTAL / (REF_TOTAL + ALT_TOTAL).

L'application identifie les colonnes d'échantillons (ex: Control pour le tissu sain et Tumor pour le tissu tumoral) pour comparer leurs VAFs et identifier les variants purement somatiques.

3. Modes d'annotation (AnnotSV)

  • Mode Global (Full) : Une seule ligne par SV. La colonne gène contient la liste de tous les gènes chevauchés (séparés par des points-virgules).
  • Mode Détaillé (Split) : Une ligne par gène chevauché, permettant d'analyser précisément l'impact pour chaque gène (exons touchés, localisation intron/exon, transcript, etc.).

4. Fonctionnalités de filtrage

Vous pouvez filtrer vos variants structuraux selon le type de SV (délétion, duplication, inversion, translocation), le chromosome, des seuils de VAF dans le tissu sain ou tumoral, la taille du variant, la classe de pathogénicité ACMG, ou en recherchant un gène spécifique.

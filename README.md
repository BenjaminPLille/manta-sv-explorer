# Manta SV Explorer

Manta SV Explorer is a browser-based application designed to facilitate the exploration and visualization of structural variants identified by Manta.

The tool was initially developed to inspect Manta output files generated by the SEQOIA laboratory and to provide a more accessible overview of detected structural variants.



## Online application



Manta SV Explorer can be used directly from a web browser:



**[Launch Manta SV Explorer](https://benjaminplille.github.io/manta-sv-explorer/)**



No installation is required. Select a compatible Manta output file from the application interface to begin exploring structural variants.



The application runs locally in the browser. Genomic files are not intentionally uploaded to an external server.



## Purpose

Structural variant output files can be large and difficult to review manually. Manta SV Explorer provides an interactive interface to:

* Import Manta structural variant results
* Explore detected deletions, duplications, inversions, insertions, and breakends
* Filter variants according to genomic and technical criteria
* Search for genes potentially affected by structural variants
* Review individual variants in greater detail
* Export selected or filtered results

## Usage

Open the application in a modern web browser and upload a compatible Manta output file.

The analysis is performed locally in the browser. Uploaded files are not intentionally transmitted to an external server by the application.

## Project status

This project is exploratory and under active development.

It is intended as a research and data-exploration tool and should not be used as a standalone clinical interpretation or diagnostic system. Results should always be reviewed alongside the original bioinformatics output and validated using appropriate laboratory procedures.
